# Axelo 3

This is a [webagency.app](https://www.webagency.app) demo book using [svelte.dev](https://svelte.dev)

## Install

```
degit regnou/axelo3
pnpm i
```

## Development

```
pnpm story:dev
```

## Production

```
pnpm story:build
pnpm story:preview
```

## Cleanup

To remove `node_modules` and `build` files

```
pnpm clean
```
